# 07-Versions

Control de versiones y releases del proyecto.

## Tipos de ficheros
- `*.zip` — Releases empaquetadas del proyecto
- `CHANGELOG.md` — Historial de cambios
- `VERSION.txt` — Versión actual del proyecto

## Nomenclatura de releases
`<proyecto>_v<major>.<minor>.<patch>_<YYYYMMDD>.zip`
Ejemplo: `mi_proyecto_v1.2.0_20240315.zip`
