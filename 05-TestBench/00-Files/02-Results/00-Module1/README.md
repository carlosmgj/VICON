# 02-Results

Resultados de simulación organizados por módulo, fecha y tipo de test.

## Estructura
```
00-Module1/
└── YYYYMMDD/
    ├── integracion/    # Tests de integración
    ├── regresion/      # Tests de regresión
    └── unitarios/      # Tests unitarios
```

## Tipos de ficheros
- `*.txt` — Resultados de simulación
- `*.log` — Logs de QuestaSim/GHDL
