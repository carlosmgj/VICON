# Constant_Packages

Packages VHDL con constantes globales del proyecto.

## Tipos de ficheros
- `*.vhd` — Packages de constantes
- `<YYYYMMDD>_constants.txt` — Historial de cambios de constantes

## Convención
Compilar estos packages ANTES que cualquier otro fichero del diseño.
GHDL los analiza en este orden en el run_doxygen.bat.
