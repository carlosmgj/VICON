# style

Recursos de estilo para la documentación HTML.

## Subcarpetas
- `extra_js/` — Ficheros JavaScript adicionales (WaveDrom offline)
- `img/` — Imágenes del proyecto (logo, fotos de equipo)

## Ficheros extra_js
- `wavedrom.min.js` — Motor de WaveDrom (offline)
- `default.js` — Skin claro de WaveDrom
- `dark.js` — Skin oscuro de WaveDrom
