# doxygen

Herramientas y scripts de documentación automática del proyecto.

## Uso
Ejecutar desde esta carpeta:
```
run_doxygen.bat
```

## Estructura
```
doxygen/
├── scripts/        # Scripts Python de generación automática
├── style/          # CSS, JS y recursos del tema Doxygen
│   ├── extra_js/   # WaveDrom JS (offline)
│   └── img/        # Imágenes (logos, fotos de equipo...)
├── Doxyfile        # Configuración de Doxygen
└── run_doxygen.bat # Script principal de generación
```
