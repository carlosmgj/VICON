# 00-Reports

Reportes generados por las herramientas EDA.

## Tipos de ficheros
- `*.rpt` — Reportes de Vivado (timing, utilización, DRC, metodología)
- `*.txt` — Reportes de análisis estático (GHDL, VSG)
- `*.csv` — Reportes de utilización exportados en formato tabla

## Nomenclatura recomendada
`YYYYMMDD_<tipo>_report.<ext>`
Ejemplo: `20240315_timing_report.rpt`
