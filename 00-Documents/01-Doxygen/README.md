# Doxygen

Documentación HTML generada automáticamente por el script `doxygen/run_doxygen.bat`.

## /Docs

### IMPORTANTE
No editar el contenido de esta carpeta manualmente.
Para regenerar la documentación ejecutar: `doxygen/run_doxygen.bat`

### Contenido generado
- `html/` — Documentación HTML navegable
- `html/index.html` — Punto de entrada de la documentación
