# ExtraContent

Contenido extra escrita de forma NO automática que se incluirá en la documentación de Doxygen como nuevas páginas o info extra de cada entidad.


## Tipos de ficheros
- `mainpage.dox` — Página principal de la documentación
- `setup.dox` — Guía de instalación y configuración del entorno
- `*.dox` — Páginas adicionales de documentación

## Ficheros auto-generados (no editar)
- `hierarchy.dox` — Generado por generate_hierarchy.py
- `reports.dox` — Generado por generate_reports.py
- `builddate.dox` — Generado por run_doxygen.bat
